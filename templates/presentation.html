Permasalahan yang diselesaikan dalam proyek ini

Objective: Membangun sistem klasifikasi gambar otomatis yang dapat membedakan antara gambar kucing dan anjing dengan akurasi tinggi menggunakan teknik Deep Learning.

Proyek ini mengimplementasikan model Computer Vision untuk klasifikasi biner (binary classification) menggunakan Transfer Learning dengan arsitektur MobileNetV2 yang telah dilatih pada dataset ImageNet.

‚úÖ Mengapa MobileNetV2?
  • Lightweight: Hanya 14MB, cocok untuk deployment
  • Efficient: Inference time cepat (~50ms per image)
  • Mobile-Friendly: Optimal untuk perangkat mobile
  • Proven Performance: Pre-trained on ImageNet (1.4M images)
Parameter Value
Base Architecture MobileNetV2
Input Size 160 x 160 pixels
Pre-trained Weights ImageNet
Training Strategy Two-Phase (Feature Extraction + Fine-tuning)
Initial Epochs 10 (Frozen Base)
Fine-tune Epochs 15 (Last ~30 layers unfrozen)
Optimizer Adam (lr=1e-5 for fine-tuning)
Loss Function Binary Crossentropy

Keterbatasan model dan area pengembangan

Current Limitations

  • Model kurang akurat untuk gambar blur atau low-quality
  • Performa menurun pada sudut kamera ekstrem
  • Binary classification only (Cat vs Dog, tidak bisa detect label lain)
  • Memerlukan gambar dengan minimal 1 objek yang jelas
  • Training di Render terbatas karena RAM 512MB

Future Improvements

  • Multi-class Classification: Tambah kategori hewan lain
  • Object Detection: Detect multiple animals dalam 1 gambar
  • Model Upgrade: Try EfficientNet atau Vision Transformer
  • Data Augmentation: Tambah variasi preprocessing
  • Real-time Prediction: Implement webcam support
  • Model Compression: Quantization untuk mobile deployment
